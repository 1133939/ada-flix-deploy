## ADA-FLIX-DEPLOY
* Projeto simples que guarda o docker-compose que faz a integração de proxy entre o frontend e o backend dos seguintes projetos
    * https://github.com/1133939/ada-flix-backend
    * https://github.com/1133939/ada-flix-frontend

#### Como rodar    
* Deve copiar o arquivo docker-compose.yml ou clonar o projeto e simplesmente executar:
    * Comando: docker-compose up

## APLICACOES
* https://localhost -> frontend

#### ACESSO AO BANCO DE DADOS
* Para acessar o banco de dados é necessário acessar a url:
    * http://localhost:8080/h2-console
    * JDBC URL: jdbc:h2:mem:mydb
    * username: sa
    * password: password
